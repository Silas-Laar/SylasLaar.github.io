03 — Work

Research Experience & Projects

Parkinson's Disease Classification Using Explainable Artificial Intelligence

Developed a fine-tuned deep learning pipeline based on VGG16 to classify Parkinson's Disease from PET scans, enabling automated differentiation between PD and non-PD cases. The pretrained network was adapted through transfer learning, with selective layer freezing and hyperparameter optimization to improve feature extraction from high-dimensional neuroimaging data. Data preprocessing included normalization, resizing, and augmentation to enhance model robustness and generalization across varying scan conditions. To interpret the model’s I applied Grad-CAM and Class Activation Mapping. These methods were applied using both single-layer and multi-layer activation mapping to generate heatmaps highlighting the most influential brain regions contributing to classification decisions.

Unsupervised Deep Learning for Parkinson's Voice Trajectory Modeling & Subtype Discovery

Developed an unsupervised deep learning framework using a Variational Autoencoder on the Oxford Parkinson's Telemonitoring Dataset to learn latent representations of voice biomarkers. The model captured hidden disease progression patterns through latent space embedding, followed by clustering to identify distinct Parkinson’s phenotypes. Temporal trajectory analysis and visualization of latent dynamics provided interpretable insights into disease evolution over time from speech-derived features.

Inter-Hemispheric Asymmetry and Functional Connectivity in Alzheimer's EEG

Investigated inter-hemispheric asymmetry and functional connectivity in a large-scale EEG dataset comprising 848,640 samples to identify disrupted brain network patterns associated with Alzheimer's Disease. Functional connectivity was quantified using Pearson correlation and Phase Locking Value (PLV). Statistical analysis and machine learning classification were applied to distinguish disease states, achieving 96% accuracy and an AUC of 0.99 .

Glioblastoma Tumor Segmentation Using Deep Learning

Developed and optimized a custom U-Net on the BraTS 2020 dataset for automated brain tumor segmentation from multimodal MRI scans. The model achieved 99.2% accuracy, an IoU of 0.82, and a Dice score of 0.61. The resulting precise tumor delineation supports earlier clinical decision-making, improved disease monitoring, and enhanced treatment planning workflows in neuro-oncology, complementing supportive care strategies such as rehabilitation and nutritional optimization.

Prediction of Inhibitors of Alpha-Glucosidase Using Machine Learning

I used machine learning to predict inhibitors and activators of the alpha-glucosidase enzyme as potential therapeutic targets for type 2 diabetes and pompe diseases. I collected data from the PubChem database on active and inactive compounds tested against the enzyme as training data and developed six algorithms comprising logistic regression, support vector machine, K-nearest neighbor, AdaBoost, random forest, and neural networks to classify compounds as active or inactive inhibitors and activators of the enzyme, which has the potential to accelerate the drug discovery process and lead to the development of more effective treatments of these diseases. Through this project, I gained expertise in data mining, machine learning, and project management.

Unsupervised Anomaly Detection in EEG Signals Using LSTM Autoencoder

Developed an LSTM Autoencoder trained exclusively on normal EEG signals to learn robust signal reconstruction patterns. Leveraged reconstruction error as an anomaly detection metric for identifying epileptic seizures in the Epileptic Seizure Recognition Dataset, achieving 94% accuracy and an ROC-AUC of 0.98. The approach enables effective detection of abnormal neural activity for early seizure identification.

Multi-Task ECG Joint Arrhythmia and Myocardial Infarction Detection

Developed a multi-task one-dimensional convolutional neural network (1D CNN) with a shared backbone that simultaneously performs five-class arrhythmia classification on the MIT-BIH dataset and binary myocardial infarction detection on the PTB dataset. The model leverages the integrated use of all four ECG signal channels to enhance generalization across heterogeneous cardiac signals and incorporates shared feature extraction with task-specific output layers to efficiently learn both rhythm abnormalities and ischemic patterns. Extensive preprocessing, including noise filtering, signal normalization, and segmentation, was applied to ensure robustness, while performance was evaluated using metrics such as accuracy, precision, recall, and ROC–AUC to validate clinical reliability.

LSTM-Based Deep Learning for Anomaly Detection in ECG Signals

Developed an LSTM Autoencoder for anomaly detection in electrocardiogram (ECG) time-series data. The model was trained exclusively on normal ECG patterns to learn accurate reconstruction behavior, with deviations quantified using reconstruction loss to identify abnormal signals. Evaluated on the ECG5000 Dataset, performance was assessed using ROC curves and AUC score to measure detection reliability and diagnostic effectiveness.

EEG Signal Processing and Machine Learning Pipeline for Alzheimer’s Disease Classification

Developed an EEG signal processing and machine learning pipeline for classification of Alzheimer's Disease, integrating advanced preprocessing and feature engineering techniques. Applied Fast Fourier Transform (FFT) band-power analysis and Haar wavelet feature extraction, alongside Independent Component Analysis (ICA) for artifact removal and wavelet-based denoising. Incorporated Principal Component Analysis (PCA) and autoencoder-based dimensionality reduction to optimize feature representation. Performed multi-classifier benchmarking on a large-scale EEG dataset comprising 848,640 samples to evaluate discriminative performance across models.
